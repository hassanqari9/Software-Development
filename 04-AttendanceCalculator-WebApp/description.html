Description

Normally attendance could be calculated using this formula

(Classes Attended / Total Classes Held) * 100

However, students need this value for two purposes

  • to calculate how many days must they attend to get their attendance upto minimum required percentage
  • to know how many days could be skipped and still not fall below minimum required percentage

Example

If a student has attended 5 classes out of 10, the attendance would be 50% but a student is usualy more interested in knowing how many more days must be attended to get the minimum required value set by the institution.

Assume 75% is the minimum required attendance then to calculate how many more days a studemt must attend, we made this calculator.

It would be incorrect to evaluate in this manner

8 / 10 * 100 = 80%
8 - 5 = 3 days

The correct method would be to increment both classes attended and total classes held until 75% is reached. If you do it manually it would look like this:

(05/10) * 100 = 50.00
(06/11) * 100 = 54.50
(07/12) * 100 = 58.30
(08/13) * 100 = 61.50
(09/14) * 100 = 64.20
(10/15) * 100 = 66.66
(11/16) * 100 = 68.75
(12/17) * 100 = 70.50
(13/18) * 100 = 72.20
(14/19) * 100 = 73.68
(15/20) * 100 = 75.00

thus a student needs to attend 10 classes in order to get 75%.
